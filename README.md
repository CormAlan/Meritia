## Vad är Meritia?
Meritia grundades i Stockholm med en enkel idé: om privatlärare får bättre betalt, blir undervisningen bättre. Och om mellanhänder minimeras, betalar föräldrar mindre.

Som studenter på KTH och Handels såg vi hur befintliga privatlärarföretag fungerade: de tog enorma marginaler, betalade privatlärare dåligt och levererade medelmåttig kvalitet.

Meritia föddes ur övertygelsen att det finns ett bättre sätt. Genom att anställa uteslutande från KTH, Karolinska Institutet och Handelshögskolan säkerställer vi akademisk excellens. Genom att minimera vår marginal ger vi privatlärare branschledande ersättning, och föräldrar ett bättre pris.

Idag är Meritia Stockholms mest exklusiva privatundervisningstjänst, byggd på principen att kvalitet aldrig ska kompromissas.
